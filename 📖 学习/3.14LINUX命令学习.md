# Linux 命令学习笔记

> 学习日期：2026-03-14[[2026-03-14]]

## 基础命令

| 命令 | 全称 | 意思 | 用途 |
|------|------|------|------|
| pwd | Print Working Directory | 打印当前工作目录 | 查看自己在哪 |
| ls | list | 列出 | 查看当前文件 |
| cd | Change Directory | 切换目录 | 进入文件夹 |
| mkdir | Make Directory | 创建目录 | 新建文件夹 |
| rm | Remove | 删除 | 删除文件/文件夹 |
| cp | Copy | 复制 | 复制文件 |
| mv | Move | 移动/重命名 | 移动或改名 |
| cat | Concatenate | 拼接 | 查看文件内容 |
| echo | 回显 | 打印输出 | 输出文字 |

## 系统命令

| 命令 | 全称 | 意思 | 用途 |
|------|------|------|------|
| uname | Unix Name | 系统名称 | 查看系统信息 |
| df | Disk Free | 磁盘剩余空间 | 查看磁盘使用 |
| free | 自由/空闲 | 内存空闲情况 | 查看内存使用 |
| top | Table of Processes | 进程表格 | 查看运行进程 |

## 进阶命令

| 命令 | 全称 | 意思 | 用途 |
|------|------|------|------|
| grep | Global Regular Expression Print | 全局正则表达式打印 | 搜索内容 |
| chmod | Change Mode | 改变权限模式 | 修改文件权限 |
| chown | Change Owner | 改变文件所有者 | 修改文件归属 |

## 巧记方法

- **pwd** = "我在哪" → Print Working Directory
- **cd** = "去哪" → Change Directory  
- **ls** = "看有啥" → list
- **cat** = "看文件" → concatenate（拼接显示）
- **df** = "磁盘free" → Disk Free
- **free** = "内存free" → 空闲内存
