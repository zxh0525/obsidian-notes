# 股票团队 - 完整工作流

> 最后更新：2026-03-26

---

## 模式：条件单 + 集合竞价精准买入

核心思路：
- 今晚选股，设定止损/止盈
- 明早集合竞价，根据开盘价精准决策买入价
- 用户在同花顺挂条件单，9:15自动触发

---

## 三阶段完整流程

### 第一阶段：今晚选股（15:00-21:00）

**目标：为明天选出候选股，设定持仓止损止盈**

1. **决策官派发任务**
   - 6研究员并行：基本面|行业|消息|技术|情绪|大盘

2. **研究员并行研究**
   - 各自使用腾讯API + Yahoo Finance研究（数据源太少，同花顺，东方财富等都可以用，最好一个分析师用一个）
   - 输出：候选股名单 + 分析理由

3. **决策官数据验证**
   - 腾讯API逐一验证股价、代码
   - 错误数据剔除，正确数据纳入

4. **多空辩论**
   - 多头研究员 vs 空头研究员针对候选股辩论

5. **交叉验证**
   - ≥3个研究员看多 → 强烈推荐
   - 2个研究员看多 → 推荐
   - <2个 → 不推荐

6. **决策官输出操作计划**
   - 大盘判断
   - **持仓股**：止损价 / 止盈价
   - **候选股**：代码 + 名称 + 买入理由 + 止损位 + 止盈位

7. **用户执行**
   - 在同花顺挂好**持仓股**的止损/止盈单（可以加上候选股的单）
   - 候选股**挂单**，等明早集合竞价（后选股也挂，竞价后可以做调整）

---

### 第二阶段：明早集合竞价（9:15-9:25）

**目标：根据开盘价精准决策买入价**

1. **9:15 集合竞价开始**
   - 决策官获取候选股开盘价
   - 用户在同花顺观察开盘情况

2. **决策官判断**
   - 候选股开盘价在买入区间？→ 符合条件
   - 候选股开盘价高于买入区间？→ 不符合，放弃
   - 候选股开盘价低于买入区间？→ 注意可能是弱势，谨慎

3. **符合条件 → 立刻告知**
   - "XXX股，开盘价XX，符合条件，现在挂买单，价格XX-XX"
   - 用户在2分钟内完成挂单

4. **不符合条件**
   - 告知原因
   - 继续观察下一只候选股
   - 无合适标的 → 今日不操作

5. **未持仓股处理**
   - 如果持仓股开盘未触发止损/止盈
   - 决策官判断是否需要手动卖出

---

### 第三阶段：持仓管理（9:30-15:00）

**目标：管理现有持仓，追踪新仓**

1. **成交确认**
   - 检查新仓是否成交
   - 更新 portfolio.json

2. **新仓设保**
   - 买入后立刻在同花顺设好止损/止盈单

3. **全天监控**
   - 触发止损/止盈 → 同花顺条件单自动成交
   - 异常情况 → 决策官实时通知

4. **收盘后**
   - 记录当日交易
   - 更新持仓记录

---

## 风控规则

- **止损位**：买入价 -5% 到 -8%
- **止盈位**：买入价 +10% 到 +15%
- **仓位控制**：单只不超过总资金30%（这个没有关系，如果单支股票股价为9元，但是非常强势，也可以买入）
- **每日最大亏损**：总资金-5%，停止当日交易
- **T+1规则**：今日买，明日才能卖

## 数据源（数据源太少，分析员需要每人最少使用两个不同的数据源）

- 实时行情 → 腾讯API（qt.gtimg.cn）
- 历史K线 → Yahoo Finance
- 研究参考 → 东方财富、同花顺
