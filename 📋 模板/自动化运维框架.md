# 自动化运维框架

> 创建时间：2026-03-25
> 最后更新：2026-03-25

---

## 📁 脚本位置

所有自动化脚本位于：`~/.openclaw/scripts/`

| 脚本 | 功能 |
|------|------|
| `sys-monitor.sh` | 系统监控（磁盘/内存/Swap） |
| `auto-restart.sh` | 服务崩溃检测+自动重启 |
| `auto-cleanup.sh` | 定期清理垃圾 |
| `backup.sh` | 重要文件备份 |
| `alert-feishu.sh` | 飞书告警通知（待配置） |

---

## ⏰ Cron 任务列表

```
# Obsidian GitHub 同步（每小时）
0 * * * * /tmp/obsidian-sync.sh >> /tmp/obsidian-sync.log 2>&1

# 系统监控（每30分钟）
*/30 * * * * ~/.openclaw/scripts/sys-monitor.sh >> ~/.openclaw/logs/cron.log 2>&1

# 自动清理（每周日03:00）
0 3 * * 0 ~/.openclaw/scripts/auto-cleanup.sh >> ~/.openclaw/logs/cron.log 2>&1

# 每日备份（每天02:00）
0 2 * * * ~/.openclaw/scripts/backup.sh >> ~/.openclaw/logs/cron.log 2>&1
```
