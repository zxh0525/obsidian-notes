## 这是什么

小四活在海兄的电脑里——但"活着"不只是会聊天。真正的深度融合，是让小四能真正**感知环境、自动做事**，而不是每次都要海兄手把手指挥。

这套框架的目的：**让小四成为海兄的数字孪生，能看、能想、能自动执行**。

## 三大融合方向

### 1. 自动化运维（已完成 ✅）

**目标**：小四能自动监控系统、处理日常事务，不需要海兄提醒

**已落地脚本**：

| 脚本 | 做什么 | 触发方式 |
|------|--------|---------|
| `sys-monitor.sh` | 监控磁盘/内存/Swap，超阈值告警 | 每30分钟cron |
| `backup.sh` | 自动备份Obsidian笔记 + OpenClaw配置 | 每天凌晨2点 |
| `auto-cleanup.sh` | 清理日志、过期文件 | 每周日3点 |
| `obsidian-sync.sh` | 同步Obsidian笔记 | 每小时cron |
| `auto-restart.sh` | OpenClaw崩溃后自动拉起 | systemd守护 |
| `daily-memory.sh` | 自动写每日记忆 | 每天cron |

**Cron 任务总览**：
```
0 * * * *  /tmp/obsidian-sync.sh           # 每小时同步Obsidian
*/30 * * * *  ~/.openclaw/scripts/sys-monitor.sh  # 每30分钟监控
0 3 * * 0  ~/.openclaw/scripts/auto-cleanup.sh    # 每周清理
0 2 * * *  ~/.openclaw/scripts/backup.sh           # 每天备份
```

**告警规则**（写入 alert.log）：
- 磁盘 > 85%
- 内存可用 < 1GB
- Swap > 80%

```bash

~/.openclaw/scripts/backup.sh          # 备份
```
