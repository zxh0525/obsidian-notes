# 晚间工作流程

> 更新日期：2026-03-25

---

## 目的
每天晚间制定第二天操作计划，确保第二天开盘前有明确策略。

---

## Step 1：制定明天卖出策略

### 持仓股处理
根据当天收盘情况，制定持仓股的明天卖出策略：

| 明日竞价情况  | 操作                             |
| ------- | ------------------------------ |
| 高开 3%+  | 止盈卖出                           |
| 高开 0-3% | 观察15分钟，如果不涨就卖（**不是等冲高，是等一等看**） |
| 平开      | 开盘观察，如果快速下跌就卖；如果稳定可以在尾盘决定      |
| 低开      | 跌破止损价立刻割，**我会提前发止损价提醒你**       |

### 计算预期现金
- 预计卖出哪些股？ → **第二天全部卖出**
- 卖出后能拿到多少现金？ → **用当天收盘持仓市值估算**

---

## Step 2：根据总资产选第二天股票

### 选股标准
- **价格**：≤ 10元
- **涨幅**：3-7%（不追高、不碰涨停板）
- **成交额**：≥ 3亿（流动性好）
- **基本面**：有实质利好或板块轮动

### 仓位控制
- 单只股票：不超过总资金 **30%**
- 现金储备：**不留**（你说的对，1000元本金小，全进去效率更高）
- 同时持有：**2-3只**（总仓位占资金 80-90%）

### 候选股票池
每天从全市场扫描，筛选符合条件的股票，优先考虑：
- 板块轮动中的补涨机会
- 昨日涨停股的跟风股（**昨天涨停的，今天回调时买**）
- 缩量回调的强势股

---

## Step 3：制定第二天开盘买入策略

| 竞价情况 | 操作 |
| ------- | ---- |
| 低开 2-3% | 挂单买入（挂比现价低2-3%的价格，等回调成交） |
| 高开 | 等10分钟看是否回落，如果一直涨到+3%以上就算了 |
| 竞价涨停 | 放弃，不追（**这种情况直接换下一只候选股**） |
| 价格稳定 | 开盘30分钟内不出现大涨大跌 = 稳定，可以买 |

**问：高开一直涨到涨停怎么办？**
答：**不追，重新选股**。A股3000只股票，错过一只还有下一只。

---

## 风险控制

- **止损位**：买入价 -8% 到 -10%（**控制到-3%太严格，容易被洗出去**）
- **止盈位**：买入价 +15% 到 +20%
- **每日最大亏损**：总资金 -5%，达到后停止当日交易
- **T+1规则**：当天买的股当天不能卖

---

## 注意事项

1. 宁可不买，也不要追高
2. 错过不亏钱，但追高会亏钱
3. 严格执行止损纪律
4. 不要全仓进出
