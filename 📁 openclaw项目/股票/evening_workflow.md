# 晚间工作流程

> 更新日期：2026-03-25

---

## 目的
每天晚间制定第二天操作计划，确保第二天开盘前有明确策略。

---

## Step 1：制定明天卖出策略

### 持仓股处理
根据当天收盘情况，制定持仓股的明天卖出策略：

| 明日竞价情况 | 操作 |
| ------- | ---- |
| 高开 3%+ | 止盈卖出 |
| 高开 0-3% | 观察15分钟，如果不涨就卖（**不是等冲高，是等一等看**） |
| 平开 | 开盘观察，如果快速下跌就卖；如果稳定可以在尾盘决定 |
| 低开 | 跌破止损价立刻割，**我会提前发止损价提醒你** |

### 明日卖出挂单设置
- **晋控电力**：今晚在券商APP挂"**明日止损价 3.55**"的卖出单（成本价3.945，跌到3.55自动触发）
- **太阳能**：今晚挂"**明日止损价 5.65**"的卖出单
- **条件单功能**：大部分券商APP支持"条件单"，设置好价格后明天自动触发，无需盯盘
- **备选方案**：如果券商不支持条件单，明天9:15集合竞价时手动挂单

### 计算预期现金
- 预计卖出哪些股？ → **第二天全部卖出**
- 卖出后能拿到多少现金？ → **持仓100股 × 明日开盘价 = 大概1050-1070元**

## Step 2：根据总资产选第二天股票

### 选股标准
- **价格**：≤ 10元
- **涨幅**：3-7%（不追高、不碰涨停板）
- **成交额**：≥ 3亿（流动性好）
- **基本面**：有实质利好或板块轮动

### 仓位控制
- 单只股票：不超过总资金 **30%**
- 现金储备：**不留**（你说的对，1000元本金小，全进去效率更高）
- 同时持有：**2-3只**（总仓位占资金 80-90%）

### 候选股票池
每天从全市场扫描，筛选符合条件的股票，优先考虑：
- 板块轮动中的补涨机会
- 昨日涨停股的跟风股（**昨天涨停的，今天回调时买**）
- 缩量回调的强势股

## Step 2.5：多空辩论（关键环节！不可跳过）

### 辩论流程
1. 决策官汇总5路研究员报告，筛选出2-3只候选股
2. 派发给多头研究员（stock_bull）：为候选股辩护
3. 派发给空头研究员（stock_bear）：挖掘做空风险
4. 多空双方充分辩论1-2轮
5. 决策官综合判断，给出最终推荐

### 多头研究员任务
- 列出买入的3个最强理由
- 反驳空头可能的论点

### 空头研究员任务
- 列出做空的3个最强风险
- 挑战多头的乐观预期

### 辩论输出格式
```
多头论点：
1. [理由]
2. [理由]
3. [理由]

反驳空头：
- 空头说... → 多头答...

空头论点：
1. [风险]
2. [风险]
3. [风险]

挑战多头：
- 多头认为... → 空头认为...
```

## Step 4：制定第二天开盘买入策略

| 竞价情况 | 操作 |
| ------- | ---- |
| 低开 2-3% | 挂单买入（挂比现价低2-3%的价格，等回调成交） |
| 高开 | 等10分钟看是否回落，如果一直涨到+3%以上就算了 |
| 竞价涨停 | 放弃，不追（**这种情况直接换下一只候选股**） |
| 价格稳定 | 开盘30分钟内不出现大涨大跌 = 稳定，可以买 |

**问：高开一直涨到涨停怎么办？**
答：**不追，重新选股**。A股3000只股票，错过一只还有下一只。

---

## 风险控制

- **止损位**：买入价 -8% 到 -10%（**控制到-3%太严格，容易被洗出去**）
- **止盈位**：买入价 +15% 到 +20%
- **每日最大亏损**：总资金 -5%，达到后停止当日交易
- **T+1规则**：当天买的股当天不能卖

---

## 注意事项

1. 宁可不买，也不要追高
2. 错过不亏钱，但追高会亏钱
3. 严格执行止损纪律
4. 不要全仓进出
