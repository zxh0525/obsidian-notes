## 二、核心策略：连板优先

### 核心理念
> **买股是为了赚钱。不是为了T+1而T+1。**

- 如果个股具备连续涨停条件，可以持有2-3天
- 每天收盘后复盘，判断明日是"继续持有"还是"卖出"
- 连板失败（开板+大盘走弱）→ 立即卖出，不恋战

### 连板判断标准（每晚复盘评估）

| 条件 | 权重 | 说明 |
|------|------|------|
| 封板坚决程度 | ★★★ | 封单量是否持续放大？是否有开板？ |
| 板块催化持续性 | ★★★ | 政策/消息是否持续发酵？还是一次性利好？ |
| 主力资金流向 | ★★★ | 超大单是否持续净流入？ |
| 大盘配合度 | ★★ | 明日大盘是否支持连板？ |
| 换手率健康 | ★★ | 换手率<15%为健康，>20%有出货嫌疑 |

### 连板决策规则

| 满足条件数 | 决策 | 目标收益 |
|------------|------|---------|
| 4-5个（高） | 持有，等连板 | +15%~+20% |
| 2-3个（中） | 谨慎持有，设紧止损 | +8%~+10% |
| 0-1个（低） | 次日开盘卖出 | 不贪 |

## 三、完整执行流程（10步）

### Step 1: 数据验证
- 腾讯API + Yahoo Finance 双重验证候选股股价
- 价格必须在10元以下
- 两个数据源差异必须<5%

### Step 2-3: 6路研究员并行
- spawn 6个子智能体：基本面|技术|消息|情绪|行业|大盘
- 每个研究员最少使用2个不同数据源
- **研究阶段由Python脚本执行（带超时重试），确保100%完成**

### Step 4: 多空辩论
- spawn stock_bull → 挖掘上涨逻辑
- spawn stock_bear → 揭示下跌风险
- **必须双方都返回有效报告，才能进入下一步**

### Step 5: 交易员综合研判
- spawn stock_trader → 制定交易计划
- 输出：买入/观望/卖出 + 买入价 + 止损止盈 + 时机
- **明确标注每只股票的"连板潜力"评级**

### Step 6: 风控审核
- spawn stock_risk → 评估风险等级
- 弱化评估：持仓涨幅 < 大盘涨幅-1% → 立即止损

### Step 7: PM最终审批
- spawn stock_pm → 最终批准/拒绝
- 输出完整"明日操作计划"
- **持仓股给出连板持有/卖出建议**

### Step 8: 交付用户（飞书群）
- 包含：大盘判断 + 候选股名单 + 止损止盈 + **连板潜力评级**

### Step 9: 用户挂单（用户手动）
- 明早9:15前在同花顺挂好条件单

### Step 10: 持仓管理（收盘后复盘，每日15:00执行）
- 检查今日持仓涨跌情况
- **连板评估**：判断明日是否继续涨停/走弱
- 做出"明日持有/卖出/加仓"决策
- 若触发弱化条件，次日立即止损

| 红线 | 阈值 |

| 连板失败（开板+大盘弱） | 立即卖出，不恋战 |

## 五、资金管理

```
可用资金池 = 现金 + (弱仓市值 × 80%)
```

**弱仓定义（满足任意一条）：**
- 持仓涨幅 < 0
- 持仓涨幅 < 大盘涨幅 -1%
- 持仓RSI(3) > 80（超买且开始走弱）
- 持有 > 3日且未盈利

## 六、涨停股规则

| 类别 | 今日操作 | 明日操作 |
|------|---------|---------|
| 已有持仓涨停 | 评估连板潜力 | 4-5条件→持有等连板；≤3条件→卖出 |
| 预判明日涨停→今日尾盘买入 | 14:30前决策买入 | 冲高评估连板，不板就走 |
| 已涨停封板→买不进去 | 放弃，等明日开板 | 评估是否回调买入 |

## 九、团队执行规范

### 子智能体调度规则
- **所有子智能体必须返回有效报告，才能进入下一步**
- 超时失败 → 立即重试（最多3次，每次间隔10秒）
- 3次仍失败 → 记录问题，报告用户，由主智能体判断是否继续

### cron自动触发
- 每个交易日**15:00**自动触发收盘选股流程
- 触发命令：cron向stock_decider发送"收盘选股"消息
- 飞书群公告：最终操作计划自动发送到群

### 弱化评估机制（持仓期间）
每次持仓检查时，评估以下3个条件，若触发≥2条→**立即卖出**：

| 条件 | 阈值 |
|------|------|
| ① 相对大盘走弱 | 持仓涨幅 < 大盘涨幅 -1% |
| ② 放量下跌 | 成交量 > 昨日1.5倍 AND 股价不涨 |
| ③ 破5日线 | 盘中实时判断收盘价跌破MA5 |
