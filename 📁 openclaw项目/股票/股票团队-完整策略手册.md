## 一、团队架构（4层分工）

### 批注说明：为什么要加 PM 角色？

> 决策官不就是 PM 吗？给我一个加 PM 角色的解释。

**回答**：决策官（我）和 PM 是分开的：
- **决策官**：协调整个团队、汇总报告、发出指令
- **PM（组合经理）**：独立第三只眼，做最终审批，防止决策官情绪化或遗漏风控

PM 像是"第二层风控"——决策官说"买"，PM 说"批准/拒绝"。两者独立，PM 可以推翻决策官。

## 二、核心策略

### 核心理念

> **买股是为了赚钱。** 不是为了 T+1 而 T+1，不是因为"不持股票过节"而空仓。

**策略逻辑（每晚8点执行）：**
1. **先判断持仓**：今晚持有的股票，明天有没有连续涨停/连续上涨的概率？
   - 有 → 继续持有
   - 没有 → 给出明天必须卖出的价格
2. **再选新仓**：结合大盘和候选股，选出明天要买的股票
3. **不为了过节而空仓**：有连板潜力就持有，没有就清仓

## 三、每日流程（每晚8点）

### 流程顺序（重要！）

```
Step 1: Python脚本 → 数据验证 + 6研究员并行研究（100%完成率）
    ↓
Step 2: 交易员 → 先评估持仓股的连板潜力
    ↓
Step 3: bull + bear → 多空辩论（必须双方都返回有效报告）
    ↓
Step 4: 风控官 → 评估风险
    ↓
Step 5: PM → 最终审批
    ↓
Step 6: 输出完整"明日操作计划"到飞书群

### Step 7: 次日竞价执行（9:15-9:25 自动执行）
- 9:15前：根据昨晚持仓分析 + 候选股名单，在同花顺挂好条件单
- 持仓股触发卖出 → 立即挂单 → 卖出后资金 → **立即按昨晚候选股执行买入**（不等确认）
- 预选股：9:15-9:20集合竞价下单，不等15分钟
```

### 持仓股处理

| 明日竞价情况 | 操作 |
| ------------ | ---- |
| 高开 3%+ | 止盈卖出 |
| 高开 0-3% | 观察5分钟，走弱立即卖；强势则持有 |
| 平开 | 开盘观察，快速下跌立即卖；稳定则在9:30前决定 |
| 低开 | 跌破止损价立刻割，不犹豫 |

**持仓触发卖出后 → 立即自动执行新仓买入流程，不等用户确认（昨晚已选定候选股）**

### 预选股买入时机（修正）

- 预选股在昨晚已确定，**9:15-9:20集合竞价是最佳买入窗口**
- 高开：9:15-9:20下单，不等
- 平开/低开：9:25前决定是否买入
- **不再等15分钟，15分钟是用来判断持仓股，不是预选股买入时机**

### 明日选股标准

| 标准 | 要求 |
|------|------|
| 价格 | ≤ 10元 |
| 涨幅 | 3-7%（不追高、不碰涨停板） |
| 成交额 | ≥ 3亿（流动性好） |
| 基本面 | 有实质利好或板块轮动 |

## 四、连板判断标准

### 批注说明

> 连板优先，不纯做T+1（并不是连板优先，我们会在每天晚上8点进行选股，一是复盘今天的股票，二是选出明天的股票，如果我们在复盘时发现现在持有的股票有连续涨停，或连续上涨的概率，那就继续持有，如果没有的话，那就给出明天卖出的价格，并选出明天买入的股票）

**已采纳**。每晚8点，复盘+选股同时进行，不是单纯追连板。

---

### 连板评估条件

| 条件 | 权重 | 说明 |
|------|------|------|
| 封板坚决程度 | ★★★ | 封单量是否持续放大？是否有开板？ |
| 板块催化持续性 | ★★★ | 政策/消息是否持续发酵？还是一次性利好？ |
| 主力资金流向 | ★★★ | 超大单是否持续净流入？ |
| 大盘配合度 | ★★ | 明日大盘是否支持连板？ |
| 换手率健康 | ★★ | 换手率<15%为健康，>20%有出货嫌疑 |

### 连板决策

| 满足条件数 | 决策 | 目标收益 |
|------------|------|---------|
| 4-5个（高） | 持有，等连板 | +15%~+20% |
| 2-3个（中） | 谨慎持有，设紧止损 | +8%~+10% |
| 0-1个（低） | 次日开盘卖出 | 不贪 |

| 红线 | 阈值 |

| 连板失败（开板+大盘弱） | 立即卖出，不恋战 |

## 六、资金管理

```
可用资金池 = 现金 + (弱仓市值 × 80%)
```

**弱仓定义（满足任意一条）：**
- 持仓涨幅 < 0
- 持仓涨幅 < 大盘涨幅 -1%
- 持仓RSI(3) > 80（超买且开始走弱）
- 持有 > 3日且未盈利

## 七、涨停股规则

| 类别 | 今日操作 | 明日操作 |
|------|---------|---------|
| 已有持仓涨停 | 评估连板潜力 | 4-5条件→持有等连板；≤3条件→卖出 |
| 预判明日涨停→今日尾盘买入 | 14:30前决策买入 | 冲高评估连板，不板就走 |
| 已涨停封板→买不进去 | 放弃，等明日开板 | 评估是否回调买入 |
