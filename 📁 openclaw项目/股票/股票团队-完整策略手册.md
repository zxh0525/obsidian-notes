## 一、团队架构（4层分工）

### 批注说明：PM 和风控官的独立性

> PM审批独立性问题的解释。

**实际情况**：PM（组合经理）= 决策官本身，不是独立Agent。

**因此**：
- PM 审批 = 我审我自己，没有真正的独立性
- **风控官（stock_risk）是独立运行的真实子智能体**
- **风控官的否决意见具有实质效力**——风控官说"不推荐"时，决策官必须给出充分理由才能推翻

决策官说"买"，风控官说"不推荐" -> **默认听风控官的**。

分析师团队(6) → 交易员（先判断持仓） → 多空辩论(2) → 风控官（有否决权）→ 决策官最终拍板

## 二、核心策略

### 核心理念

> **买股是为了赚钱。** 不是为了 T+1 而 T+1，不是因为"不持股票过节"而空仓。

**策略逻辑（每晚8点执行）：**
1. **先判断持仓**：今晚持有的股票，明天有没有连续涨停/连续上涨的概率？
   - 有 → 继续持有
   - 没有 → 给出明天必须卖出的价格
2. **再选新仓**：结合大盘和候选股，选出明天要买的股票
3. **不为了过节而空仓**：有连板潜力就持有，没有就清仓

## 三、每日流程（每晚8点）

### 流程顺序（重要！）

```
Step 1: Python脚本 → 数据验证 + 6研究员并行研究（100%完成率）
    ↓
Step 2: 交易员 → 先评估持仓股的连板潜力
    ↓
Step 3: bull + bear → 多空辩论（必须双方都返回有效报告）
    ↓
Step 4: 风控官 → 评估风险
    ↓
Step 5: 风控官（有否决权）+ 决策官最终拍板
    ↓
Step 6: 输出完整"明日操作计划"到飞书群

### Step 7: 次日竞价执行（9:15-9:25 自动执行）
- 9:15前：根据昨晚持仓分析 + 候选股名单，在同花顺挂好条件单
- 持仓股触发卖出 → 立即挂单 → 卖出后资金 → **立即按昨晚候选股名单执行买入（2选1，哪个能买买哪个）**
- 预选股：9:15-9:25观察，9:25-9:30确认后下单
```

### 持仓股处理

| 明日竞价情况 | 操作 |
| ------------ | ---- |
| 高开 3%+ | 止盈卖出 |
| 高开 0-3% | 观察5分钟（9:15-9:20观察），走弱立即卖；强势则持有 |
| 平开 | 9:20-9:25观察，快速下跌立即卖；强势则在9:30前决定 |
| 低开 | 跌破止损价立刻割，不犹豫 |

**持仓触发卖出后 → 立即按昨晚候选股名单执行买入（2选1：哪个明天能买就买哪个）**

具体规则：
- 持仓卖出后 → 资金到账 → 立即检查昨晚候选股中哪只有买入窗口
- 若两只都能买（价格合理）→ 优先买昨晚最强的那只
- 若只有一只有买入窗口（另一只涨停/买不进）→ 买能买的那只
- 若两只都买不进 → 空仓等回调，不强行追高

### 预选股买入时机（修正）

修正：9:15-9:20是虚假申报阶段，价格可能失真，不要急着下单。

- 9:15-9:20：观察竞价方向（只看不买）
- 9:20-9:25：根据竞价走势判断是否下单
- 9:25-9:30：确认最终买入价格，按昨晚计划执行
- **高开>3%：一律不追，等回调**
- 平开/低开：9:25-9:30确认后买入

### 明日选股标准

| 标准 | 要求 |
|------|------|
| 价格 | ≤ 10元 |
| 流通市值 | 10-80亿（过滤微盘股和超大盘） |
| 涨幅 | 3%-8%的首板股（明确排除涨停板） |
| 成交额 | 0.5亿-3亿（与本金和短线策略匹配；>3亿门槛太高，<0.5亿有出货风险；量价配合健康即可） |
| 基本面 | 有实质利好（认定方式见下方） |

**实质利好认定（优先级从高到低）：**
① 官方公告/监管文件/政策文件（最权威，可信度最高）
② 政策催化+成交额>0.5亿（短线有效性较高）
③ 成交额放大+板块确认（以上两条都不满足时，作为补充信号，不作为主要依据）

> 注意：利好认定不能只用"股价涨了"来验证——这是以果定因，容易被主力制造假信号欺骗。

修正：涨停板(涨幅>=9.5%)不碰；首板股(3-8%)可关注；竞价平开或低开优先。

## 四、连板判断标准

### 批注说明

> 连板优先，不纯做T+1（并不是连板优先，我们会在每天晚上8点进行选股，一是复盘今天的股票，二是选出明天的股票，如果我们在复盘时发现现在持有的股票有连续涨停，或连续上涨的概率，那就继续持有，如果没有的话，那就给出明天卖出的价格，并选出明天买入的股票）

**已采纳**。每晚8点，复盘+选股同时进行，不是单纯追连板。

---

### 连板评估条件

| 条件 | 权重 | 量化标准 |
|------|------|------|
| 封板坚决程度 | ★★★ | 封单额 > 流通市值的5%；无开板 |
| 板块催化持续性 | ★★★ | 满足以下任一即可：
① 消息已持续催化>=2日
② 首日14:00前出现：成交额>1亿+封板坚决+大盘不大幅下跌
> 注：14:00后尾盘偷袭的跟涨不纳入，主力尾盘拉升次日砸盘概率高 |
| 主力资金流向 | ★★★ | 超大单连续净流入>=2日，且净流入>1亿 |
| 大盘配合度 | ★★ | 明日大盘预判不是弱势（平/涨） |
| 换手率健康 | ★★ | <15%健康；15-20%观察；>20%警惕出货 |

注：5个条件全部量化，减少主观判断。

### 连板决策

| 满足条件数 | 决策 | 目标收益 |
|------------|------|---------|
| 4-5个（高） | 持有，等连板 | +15%~+20% |
| 2-3个（中） | 谨慎持有，设紧止损 | +8%~+10% |
| 0-1个（低） | 次日开盘卖出 | 不贪 |

## 六、资金管理（修正）

> 本金约1100元，策略以"全仓押一只最强机会"为核心，不追求分散持仓。

```
可用资金池 = 现金（不含弱仓市值）
单只仓位上限 = 80%（接受全仓押1只）
候选股上限 = 2只（最终只买1只最强）
```

**1100元本金特殊规则：**
- 单只仓位上限80%（≈880元），实际上几乎必然全仓押1只
- 候选股最多保留2只，最终只买1只，不分散
- 弱仓1日内必须决策：跌破成本价立即走，不拖延

弱仓处置原则：触发弱仓条件 -> 立即止损，不加码。亏损时越补越亏，这是"摊低成本陷阱"。

**核心逻辑：钱少的时候，只押最强的一次机会，等确定性最高的买点。**

**弱仓定义（满足任意一条）：**
- 持仓涨幅 < 0
- 持仓涨幅 < 大盘涨幅 -1%
- 持仓RSI(3) > 80（超买且开始走弱）
- 持有 > 1日且未盈利（本金小，持有时间越长风险越大）
