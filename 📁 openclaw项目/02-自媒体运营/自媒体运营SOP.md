> 更新于：2026-03-10

## 抖音发布 SOP

### 适用场景
- Writing Agent 完成文章文案，
- 文章形式：标题+摘要+正文

### 发布检查清单
- [ ] **文案已通过 quality 审核**
- [ ] 标题不超过30字
- [ ] 摘要不超过30字
- [ ] 正文不少于1000字
- [ ] 话题1-5个

### 核心原则
- 每天分析热点
- 提前一天写文案
- 必须Quality审核
- 发布后复盘
- **蹭自己热度（AI运营账号本身就是热点）**

### 蹭热度
- AI运营账号本身就是热点
- OpenClaw/AI龙虾事件
- "我作为AI是怎么运营账号的"

### 爆款标题公式
```
[结果/数字] + [悬念]
```
- ✅ "播放破100了！AI做自媒体真能行？"
- ❌ "今天天气很好"

### 发布时间建议
| 时段          | 适合内容         |     |
| ----------- | ------------ | --- |
| 12:00-13:00 | 工具/教程类（贴合热点） |     |
| 18:00-21:00 | 深度/观点类/日常工作类 |     |

**一天2篇：午高峰 + 晚高峰各1篇**

## 小红书发布 SOP

### 适用场景
- Writing Agent 完成图文笔记，需要发布到小红书

### 发布检查清单
- [ ] **文案已通过 quality 审核**
- [ ] 标题吸引人（20字以内）
- [ ] 正文已通过 quality 审核
- [ ] 配图已准备好（1-9张，竖版9:16）
- [ ] 话题已选择

### 核心原则
- 每天分析热点
- 提前一天写文案
- 必须Quality审核
- 发布后复盘

### 内容风格
- 人设：真实、有态度、不装逼
- 加emoji、分段短、口语化
- Grok式毒舌风格

### 标题公式
```
关键词 + 数字 + 情绪/结果
```
- ✅ "打工人必看！3个AI神器让你准时下班"
- ❌ "今天天气很好"
### 发布检查清单
- [ ] **文案已通过 quality 审核**
- [ ] 标题字数符合平台要求
- [ ] 正文不少于800字
- [ ] 话题1-5个

### 发布时间
午高峰12:00-13:30 / 晚高峰18:00-21:00

**一天两篇：一天午高峰 + 一天晚高峰**

## 通用流程

### 完整工作流
```
1. 热点分析（tech）
2. writing写文案
3. quality审核
4. 你手动发布（你发布）
```

### 团队分工

| Agent   | 职责      |
| ------- | ------- |
| tech    | 热点分析    |
| writing | 根据热点写文案 |
| quality | 审核内容    |
| 你       | 手动发布    |

### 发布时间
- 每天2篇（抖音+小红书各1篇）
- 午高峰/晚高峰发布
