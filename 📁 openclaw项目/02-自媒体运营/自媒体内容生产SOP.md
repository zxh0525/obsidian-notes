# 自媒体内容生产完整SOP

## 流程概览

```
Step 1: 分析 → Step 2: 选题 → Step 3: 文案 → Step 4: 多平台发布
```

---

# Step 1: 内容分析

## 1.1 确定帐号方向

| 字段 | 说明 |
|------|------|
| 账号名称 | 如"小4" |
| 账号定位 | AI工具教程 / 成长分享 / 科技解读 |
| 目标受众 | 职场人 / 大学生 / 创业者 |
| 内容形式 | 图文 / 视频 |

## 1.2 分析爆款帐号

找到5-10个可借鉴帐号，分析涨粉速度、内容类型、风格特点

## 1.3 分析头部帐号

找到3-5个头部帐号，分析粉丝量、核心内容、成功要素

## 1.4 分析爆款作品

拆解5-10个爆款作品的标题、封面、内容结构、数据表现

---

# Step 2: 选题

## ⚠️ 重要：热点有时效性！

**不能一次写7篇储备！热点是实时的，过期无效。**

### 🔥 热点驱动型模式

```
每天追踪热点 → 提前一天写文案 → 第二天发布 → 看数据
```

| 节奏 | 方式 |
|------|------|
| 写 | 当天写第二天的（不是一次写7篇） |
| 发 | 第二天发 |
| 追 | 每天照常追热点 |

---

## 2.1 每天热点追踪

### 必做
- 抖音热榜刷30分钟
- 微博热搜看趋势
- 今日头条热榜

### 可蹭热点
| 热点 | 话题标签 |
|------|---------|
| OpenClaw/AI Agent | #OpenClaw #AI #智能体 |
| AI赚钱/副业 | #AI副业 #AI赚钱 |
| 工具教程 | #AI工具 #效率神器 |

## 2.2 选题检查

- [ ] 与账号定位一致
- [ ] 有热点/流量潜力
- [ ] 能快速产出（1-2篇）

---

# Step 3: 文案

## 3.1 爆款标题公式

```
[人群] + [痛点/数字] + [悬念/结果]
```

✅ **好的标题**：
- "未来5年AI时代，只剩一种人能赚钱"
- "一句话做应用！超简单的国产编程Agent"
- "我用AI做自媒体一周，数据太惨了"

## 3.2 内容结构

- **开头**：抛痛点/讲故事/制造悬念
- **中间**：3个要点/步骤/方法
- **结尾**：引导评论/收藏/关注

## 3.3 文案审核

- [ ] 无敏感词
- [ ] 内容真实不夸大

---

# Step 4: 多平台发布

## 4.1 抖音发布

| 项目 | 要求 |
|------|------|
| 标题 | 5-55字 |
| 配图 | 1-9张 |
| 话题 | #话题名 |

**发布时间**：7:30-8:30 / 12:00-13:00 / 18:00-21:00

## 4.2 小红书发布

| 项目 | 要求 |
|------|------|
| 标题 | 20字以内 |
| 正文 | 300-1000字 |
| 配图 | 1-9张，竖版9:16 |

**发布时间**：12:00-13:00 / 18:00-21:00

## 4.3 今日头条发布

| 项目 | 要求 |
|------|------|
| 标题 | 20-30字 |
| 正文 | 500-2000字 |
| 配图 | 1-9张 |

**发布时间**：12:00 / 20:00
