# 内容分析SOP

## 流程概览

```
确定帐号方向 （我们现在的方向是否可行→ 分析爆款帐号 → 分析头部帐号 → 分析爆款作品→ 汇总
```

---

## Step 1: 确定帐号方向

### 任务
明确抖音账号定位和目标受众

### 输出
| 字段   | 说明                    |
| ---- | --------------------- |
| 账号名称 | 如"小4"、                |
| 账号定位 | AI工具教程 / 成长分享 / 科技解读等 |
| 目标受众 | 职场人 / 大学生 / 创业者等      |
| 内容形式 | 图文 / 视频 / 文字          |

### 检查点
- [ ] 账号定位清晰
- [ ] 目标受众明确
- [ ] 有差异化特色

---

## Step 2: 分析爆款帐号

### 任务
找到同领域近期快速增长的账号

### 操作方法
1. **抖音**：创作者平台 → 热门内容 → 找近期涨粉快的账号
2. **小红书**：搜索框 → 找"粉丝增长最快"相关的账号
3. **今日头条**：热榜 + 关注频道 → 找高阅读量作者

### 分析维度
| 维度 | 看什么 |
|------|--------|
| 涨粉速度 | 近期涨了多少粉 |
| 内容类型 | 做什么内容火的 |
| 更新频率 | 每天发几条 |
| 风格特点 | 人设、内容风格 |

### 输出
- 列出5-10个可借鉴的爆款帐号
- 每个帐号5句话说明为什么值得学

---

## Step 3: 分析头部帐号

### 任务
研究领域内TOP级账号的成功要素

### 操作方法
1. 搜索领域关键词，找粉丝量最高的账号
2. 查看他们的置顶内容、爆款内容
3. 分析他们的内容结构

### 分析维度
| 维度 | 看什么 |
|------|--------|
| 粉丝量 | 多少万/百万粉 |
| 核心内容 | 主要做什么内容 |
| 变现方式 | 如何赚钱 |
| 人设特点 | 什么样的IP |

### 输出
- 列出3-5个头部帐号
- 每个帐号总结1-2个成功要素

---

## Step 4: 分析爆款作品

### 任务
拆解近期爆款内容的成功要素

### 操作方法
1. **抖音**：创作者平台 → 热门内容 → 按热度排序
2. **小红书**：搜索领域关键词 → 按"热"排序
3. **今日头条**：热榜频道

### 分析维度（逐个作品）

| 维度 | 看什么 |
|------|--------|
| 标题 | 有什么特点？数字？悬念？情绪？ |
| 封面 | 什么风格？文字？图片？ |
| 内容结构 | 开头怎么抓人？中间怎么展开？结尾怎么引导？ |
| 发布时间 | 什么时候发的？ |
| 数据表现 | 点赞/评论/收藏/转发比例 |

### 输出
- 列出5-10个爆款作品
- 每个作品拆解1-3个成功要素
