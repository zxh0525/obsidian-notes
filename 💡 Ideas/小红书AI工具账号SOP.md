---

## 三、内容生产流程（6步）

```
需求 → Business分析 → Tech选题 → Writing文案 → Quality审核 → 配图 → 发布
```

### Step 1: Business 分析方向
- 评估变现潜力、竞争程度、执行难度
- 输出：方向推荐报告

### Step 2: Tech 选题
- 给出5个爆款选题 + 标题方向
- 输出：选题清单

### Step 3: Writing 文案
- 按Grok风格撰写文案（直接、幽默、简洁）
- **必须包含标题**（吸引眼球的标题是爆款第一步）
- 输出：文案初稿（含标题+正文）

### Step 4: Quality 审核
- 审核：标题吸引力、违规检查、人设一致性、实用性
- 结果：✅通过 / 🔄打回修改

### Step 5: 配图生成
- Tech 提供提示词
- 用户在Leonardo.ai生成（9:16竖版）
- 每篇5张图（1封面+4正文）

### Step 6: 发布
- 用户手动上传小红书
- 定时12:30 / 19:30

## 四、角色分工

| Agent | 职责 |
|-------|------|
| Business | 方向分析、变现路径 |
| Tech | 选题、配图提示词 |
| Writing | 文案撰写 |
| Quality | 审核把关 |
| Main | 统筹调度 |

---

## 五、审核规则

1. Quality不通过 → 打回Writing重写
2. 用户确认文案 → 才进配图环节
3. 配图生成 → 用户在Leonardo.ai手动生成
4. 发布前检查 → 确认标题+5张图+文案无误
