# 小红书AI工具账号运营SOP

---

## 1. 账号基础信息

| 项目 | 内容 |
|------|------|
| **昵称** | 海兄玩AI |
| **方向** | AI工具教程分享 |
| **目标人群** | 想学AI但不懂编程的普通上班族、学生 |
| **人设** | 不会编程的AI玩家，文科生/打工人也能上手 |
| **内容风格** | Grok风格（直接、幽默、简洁、不说废话） |

**账号装修：**
- 头像：真人出镜或AI相关潮图
- 简介：不会编程的AI玩家｜分享AI工具教程｜让AI帮打工人效率翻倍

---

## 2. 发布节奏

| 项目 | 安排 |
|------|------|
| **发布时间** | 12:30（午高峰）+ 19:30（晚高峰） |
| **发布频率** | 每天1-2条 |
| **内容形式** | 图文60% + 视频30% + 混合10% |

---

## 3. 内容生产流程（6步）

```
Step 1: Business 分析方向
  → 输出：方向推荐报告

Step 2: Tech 选题
  → 输出：5个爆款选题 / 标题方向

Step 3: Writing 文案
  → 输出：5篇完整文案（Grok风格）

Step 4: Quality 审核
  → 输出：审核意见（通过✅ / 打回🔄）

Step 5: 配图生成
  → Tech提供提示词，用户在Leonardo.ai手动生成
  → 每篇5张图，共25张

Step 6: 发布
  → 用户手动上传，定时12:30 / 19:30
```

---

## 4. 角色分工

| Agent | 职责 | 产出 |
|-------|------|------|
| **Business** | 方向分析、变现路径规划 | 方向推荐报告、变现方案 |
| **Tech** | 选题调研、配图提示词 | 选题清单、配图提示词 |
| **Writing** | 文案撰写、标题优化 | 完整文案 |
| **Quality** | 审核把关、风险控制 | 审核意见 |
| **Main** | 统筹调度、进度追踪 | 全流程管理 |

---

## 5. 审核规则

1. **Quality审核** → 不通过则打回Writing重写
2. **用户确认文案** → 才进入配图环节
3. **配图生成** → 用户在Leonardo.ai手动生成（9:16竖版）
4. **发布前检查** → 确认5张图+文案无误
